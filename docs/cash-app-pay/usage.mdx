# Basic usage

Take a Cash App Pay payment.

```tsx
import { CashAppPay, PaymentForm } from 'react-square-web-payments-sdk';

export default function MyApp() {
  return (
    <PaymentForm>
      <CashAppPay />
    </PaymentForm>
  );
}
```

:::info

Cash App Pay is currently supported only in the US.

:::
